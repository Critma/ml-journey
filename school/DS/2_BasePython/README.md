# Base Python Exercises — DS Track

## Требования

### Упражнение 00 — Типы данных
- Файл: `data_types.py`
- Задача: объявить 8 переменных разных типов (`int`, `str`, `float`, `bool`, `list`, `dict`, `tuple`, `set`) и вывести их типы
- Ограничения: без импортов, использовать встроенные типы

### Упражнение 01 — Работа с файлами
- Файлы: `read_and_write.py`, `ds.csv`
- Задача: считать CSV-файл, заменить запятые на табуляторы (`\t`), сохранить в TSV-файл
- Особенность: обработка запятых внутри строк данных

### Упражнение 02 — Поиск по ключу
- Файл: `stock_prices.py`
- Задача: по названию компании вывести цену акции
- Словари: `COMPANIES` (название → тикер) и `STOCKS` (тикер → цена)
- Поведение: "Unknown company" для неизвестных компаний, пустой вывод при некорректных аргументах

### Упражнение 03 — Поиск по значению и по ключу
- Файл: `ticker_symbols.py`
- Задача: по тикеру вывести название компании и цену акции
- Поведение аналогично ex02

### Упражнение 04 — Словари
- Файл: `to_dictionary.py`
- Задача: преобразовать список кортежей (страна, число) в словарь с ключом-числом
- Особенность: один ключ может иметь несколько значений

### Упражнение 05 — Поиск по значению или по ключу
- Файл: `all_stocks.py`
- Задача: обработать строку с выражениями (названия компаний и тикеры), определить тип каждого выражения
- Поведение: вывести результат в формате "X is a ticker symbol for Y" или "X stock price is Z" или "X is an unknown company or an unknown ticker symbol"

### Упражнение 06 — Сортировка словаря
- Файл: `dict_sorter.py`
- Задача: вывести страны в порядке убывания по числу, при равенстве — по алфавиту
- Использовать список кортежей из ex04

### Упражнение 07 — Множества
- Файл: `marketing.py`
- Задача: выполнить операции над множествами для маркетинговых задач:
  - Клиенты, не видевшие рекламное письмо (для колл-центра)
  - Участники, не являющиеся клиентами (для приветственного письма)
  - Клиенты, не участвовавшие в мероприятии (для отправки видео)

### Упражнение 08 — Работа со строками как со списками
- Файлы: `names_extractor.py`, `letter_starter.py`
- Задача `names_extractor.py`: извлечь имя, фамилию и email из почтовых адресов формата `name.surname@corp.com`, сохранить в TSV
- Задача `letter_starter.py`: по email найти имя и вернуть приветственное письмо

### Упражнение 09 — Шифр Цезаря
- Файл: `caesar.py`
- Задача: зашифровать или расшифровать строку с помощью шифра Цезаря
- Поведение: поддержка команд `encode` и `decode`, сдвиг по алфавиту
- Ограничение: выброс исключения для кириллических символов

| Упражнение | Файлы | Описание |
|------------|-------|----------|

| ex01 | `read_and_write.py` | Преобразование CSV в TSV с обработкой запятых внутри строк|

| ex03 | `ticker_symbols.py` | Поиск названия компании и цены по тикеру через инвертированный поиск в словарях |
| ex04 | `to_dictionary.py` | Преобразование списка кортежей в словарь и вывод в нужным формате |
| ex05 | `all_stocks.py` | Обработка строки с выражениями (названия/тикеры) и определение типа каждого наименования |
| ex06 | `dict_sorter.py` | Сортировка словаря по числам (убывание) и ключам (алфавит) при равных значениях |
| ex07 | `marketing.py` | Операции над множествами для решения маркетинговых задач (разность множеств) |

## Технологии

- **Set operations** - операции над множествами (разность) в ex07
- **String manipulation** - split, capitalize, f-strings в ex01, ex08
- **File I/O** - чтение и запись файлов в ex01, ex08
- **sys.argv** - обработка аргументов командной строки в ex02-ex07, ex09
- **ord/chr** - работа с кодами символов для шифрования Цезаря в ex09
